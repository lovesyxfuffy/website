中铁建设前后端接口文档
===========
### 备注:通用返回格式
response:
```json
{
    "data":{
    },
    "status":1
}
```
status=0 代表失败
status=1 代表成功
默认method=post 除非特意注明

### 后台部分

#### 1.后台登录
url: /manage/account/login

request:
```
{
    "admin":"xxxx",
    "password":"xxxxx"
}
```

response:
```
{}
```

#### 2.后台设置配置项
url: /manage/account/login

request:
```
{
    "headUrl":"xxxxxxxx",//头图链接

    "address":"xxxx",//地址

    "fax":"xxxxx"//传真
}
```

response:
```
{
    "data":"",
    "status":1
}
```

#### 3.后台上传文章（包括编辑）
ps:分支机构当做文章上传处理 标题为分支机构名称
ps2:精品工程当做文章上传处理,如果选择精品工程,多出一个表单项,叫做缩略图
url:/manage/article/upload

request:
```
{
    "id":3,//id不传为新增 传为编辑
    "title":"xxxx",
    "content":"xxxxx",
    "creator":"xxxxx",
    "imgUrl":"xxxxx",//缩略图url
    "type":"xx"//notice:通知公告,information中标信息,news新闻中心,field业务领域,department分支机构,project精品项目
}
```
response
```
{
    "data":{},
    "status":"1"
}
```
